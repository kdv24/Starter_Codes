  • Ping Pong

    • For this assignment, the applicant is asked to design a web page where the user is prompted to enter a number, and then the page displays every number up to that number. But, for multiples of three, the page prints "ping" instead of the number, and for multiples of five, the page prints "pong". For numbers that are multiples of three and five, the page prints "ping-pong".

  • Swift Ping Pong Gist

    • I chose to complete the Ping Pong challenge using the new language, Swift. Because I am currently learning Swift, it seemed like a good opportunity for me to meet the requirement for this application while also practicing my new language! My code is shown in the attached Gist.

  • The Layout Test

    • The challenge for the Layout Test is simply to make two versions of a given 2 column CSS layout. One version must be completed using Bootstrap, and the other must be independently designed using one's own CSS. In an ideal world, both are responsive, have stacked columns for mobile, and look pretty!
